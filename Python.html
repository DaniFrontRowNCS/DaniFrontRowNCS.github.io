Fonctions basiques Hello, World!

Fonctions basiques de Python :

  • print() : Affiche du texte ou des variables.
  • input() : Obtient une entrée de l'utilisateur.
  • len() : Retourne la longueur d'une séquence (liste, chaîne, etc.).
  • range() : Génère une séquence de nombres.
  • def : Définit une fonction.
  • return : Renvoie une valeur depuis une fonction.

Exemple : Hello, World! en Python

Pour afficher "Hello, World!" en Python :

print("Hello, World!")
